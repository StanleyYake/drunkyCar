# drunkyCar
一个基于道路分割技术的自动驾驶仪算法

核心思想:

1、CNN太慢，机器视觉也可以搞定道路分割，只不过太烧脑

2、智商=300-源文件行数。



般情况下只有一个App.cpp,不定期release。

仅依赖：OpenCV 3.x 带contrib



Structured Forests for Fast Edge Detection
